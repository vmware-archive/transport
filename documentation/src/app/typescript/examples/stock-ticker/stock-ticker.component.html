Transport for TypeScript Example: Stock Ticker

In this example, we have used Clarity Core, combined with React. We have created a simple StockTicker.tsx Component that renders out a form input, along with a button. Type in any known stock ticker symbol and press the button.

When you submit the form, the component uses Transport to call our live demo broker that uses plank.

plank is running a simple stock ticker service on the stock-ticker-service channel.

The provider API has a hard limit of 5 calls per minute and 500 calls per day, so you may see an error. Sorry.
