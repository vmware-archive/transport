Transport for TypeScript Example: Simple Stream

In this example, we have used Clarity Core, combined with React. We have created a simple SimpleStream.tsx Component that renders out a button that starts or stops the stream.

When you submit the form, the component uses Transport to stream from our live demo broker that uses plank.

plank is running a simple simple stream service on simple-stream-service channel. It will emit a random word on the stream, every 1 second.
