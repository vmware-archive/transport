Importing Transport for TypeScript into Angular Applications

Make sure you install transport first. Package is available via npm.


Angular provides angular.json (for Angular 7+) and angular-cli.json (for Angular 4-5).

Configuring angular.json

You need to add @vmw/transport to the allowedCommonJsDependencies option that is a part of architect/build/options property in your angular.json 

...
 "allowedCommonJsDependencies": ["@vmw/transport"],
...

Your application is ready to go, now you just need to boot transport

Angular provides a src/main.ts file, which is essentially your initialization script, that you can use to Initialize the Bus. 

import {BusUtil} from "@vmw/transport/util/bus.util";
import {LogLevel} from "@vmw/transport/log";

BusUtil.bootBus();
