Transport for TypeScript

View TypeScript source on Github

This implementation is designed to operate in the Browser (for right now anyway). However it is framework agnostic, and will work along-side any UI framework you want to implement.

Transport does not care which framework you would like to run. It's directly accessible from the window object, if you don't even want to use a framework. We're using examples from Angular and React in places, however the same principals would apply in any framework.
