Importing Transport for TypeScript via UMD

Make sure you install transport first, Package is available via npm.


UMD module import through the HTML <script> tag

If you would like to use Transport in a simple setup, say an HTML with WebComponents in it, you can import the UMD version of the library via a script tag. Note that RxJS is the only prerequisite for Transport so make sure to import it first. 

<script src="https://cdn.jsdelivr.net/npm/rxjs@6.6.3/bundles/rxjs.umd.min.js"></script>
<script src="https://cdn.jsdelivr.net/npm/@vmw/transport@latest/transport.umd.min.js"></script>
<script>
    // boot the event bus!
    transport.TransportEventBus.boot();
</script>

Try it out in StackBlitz

Importing Transport in your code

The main interface you will need is EventBus. It provides access to the most common methods. 

 import { EventBus } from '@vmw/transport';
